# 磐石指标口径字典

> 本字典定义磐石指标中心对外提供的核心指标口径，是各业务方报表、看板、自助分析取数的统一基准。如对口径有疑问，请先查本文档，再联系数据负责人苏晴（suqing01）。底层数据来源由数仓组（司南）维护，指标权限申请走账号中台（林晚）。

## 一、活跃类指标

### DAU（日活跃用户数）
- **定义**：当日（自然日 00:00:00–23:59:59，时区 UTC+8）成功登录的去重用户数。
- **口径**：以 `dwd_user_login_log` 的 `user_id` 按 `login_date` 去重计数。一个用户当日多次登录只计 1。
- **SQL 口径描述**：`SELECT login_date, COUNT(DISTINCT user_id) AS dau FROM dwd_user_login_log WHERE login_status = 'success' GROUP BY login_date`
- **排除项**：内部测试账号（`is_internal = 1`）、被封禁账号当日仍计入（历史遗留，暂未排除）。

### MAU（月活跃用户数）
- **定义**：自然月内"活跃"的去重用户数。
- **口径**："活跃"目前以**登录**为准，与 DAU 同源去重。
- **备注**：部分业务方主张 MAU 应以"有效行为"（登录 + 核心功能操作）口径统计，但当前指标中心仍按登录口径出数。两套口径并存，取数时请向需求方确认用哪一套。**（此处口径偏模糊，待统一）**

### 留存率（次日 / 7日 / 30日留存）
- **定义**：第 0 日新增用户中，在第 N 日仍登录的比例。
- **口径**：分母为 `register_date = D0` 的新增用户，分子为这批用户在 `D0 + N` 日有登录记录的去重数。
- **次日留存** = N=1，**7日留存** = N=7，**30日留存** = N=30。
- **注意**：留存采用"精确第 N 日"口径，非"N 日内任意一天"。

## 二、交易类指标

### GMV（成交总额）
- **定义**：统计周期内所有已成交订单的金额合计。
- **口径**：取 `dwd_order` 中 `order_status IN ('paid','shipped','completed')` 的 `pay_amount` 求和。
- **退款处理**：当前 GMV **未扣减退款**金额（即包含已退款订单的原始成交额）。若需要"净 GMV"，需另行在自助分析中减去退款流水。**（含退款与否口径长期有争议，以本文为准 = 不扣退款）**
- **币种**：统一折算为人民币（CNY），汇率按下单日快照。

### 客单价（ARPU 口径）
- **定义**：GMV / 成交用户数。
- **口径**：分子用上文 GMV（不扣退款），分母为周期内有成交订单的去重 `user_id`。

## 三、转化类指标

### 转化率（下单转化率）
- **定义**：访问商品详情页的用户中，最终下单的比例。
- **口径**：分母为浏览过商详页的去重 UV，分子为同一周期内产生下单行为的去重用户。
- **注意**：转化率为同周期口径，不做跨周期归因。复杂归因需联系 MaaS（赵启明）做自然语言取数或定制模型。
