# Lucius (luciusai.com) 落地页视觉拆解（参考）

> 来源：中文 https://luciusai.com/zh ，英文 https://luciusai.com/en （裸域 https://luciusai.com/ 按地区**默认重定向到 /zh**，语言客户端切换）。2026-06-27 实测抓取。
> 中英文同一套设计，仅文案不同（英文大标题 *One AI teammate across every community channel.*）。
> Lucius 是 **replica 的直接竞品**（社区/客服 AI，详见 `docs/竞品调研/Lucius.md`）；这里只看它的**落地页视觉**。
> 技术栈：站点用 **Momen**（无代码搭站平台）构建。

---

## 一、整体气质

一句话：**「暖橄榄米底 + 3D 黏土插画 + 一抹紫」** 的温暖编辑风。

- 不是科技冷色，而是**暖米/橄榄色调**，像文具/书房，传达"一个靠谱、温和的 AI 同事"。
- 主视觉是**3D 黏土质感插画**（台灯、写字板、绿色收纳盒、钢笔、渠道图标药丸），用紫色"线缆"把各渠道连到中央——把"AI 同事打理你的桌面"具象化。
- 克制留白 + 系统字体，整体**亲切、可信、不炫技**。与 matrix 的"摄影 + 衬线大字"是两条不同的高级感路线。

---

## 二、字体系统（和 matrix 思路不同，值得对比）

| 角色 | 选型 | 说明 |
|---|---|---|
| 大标题 / 正文 / UI | **系统字体栈**（`-apple-system, SF Pro Display, PingFang SC, Microsoft YaHei, Noto Sans SC`） | **没有自定义品牌字体**，全靠系统无衬线，60px/字重 400；省事、加载快、原生感强 |
| 标题里的**强调词** | **斜体衬线**（generic `serif`）+ **紫色** | 每句大标题里挑一个词做"斜体衬线 + 紫色"强调（如 *teammate* / *同事*），瞬间提气质——这是和 matrix 共享的"斜体衬线强调"trend |
| 眼标 / 编号 | 系统字 small caps 感 + 前导短横 | 如 `— How it works`、`/01 /02 /03`，弱化灰色 |

> 关键差异：**matrix = 整句衬线**（Instrument Serif），**lucius = 系统无衬线 + 单词斜体衬线强调**。后者复刻成本极低：正文用系统字/Inter，强调词包一个 `<em>` 上 `font-style:italic; font-family:Georgia/serif; color:紫`。

## 三、配色（实测值）

| 用途 | 值 | 备注 |
|---|---|---|
| 页面/区块暖底 | 橄榄米 `~#E8E6D6`、次级 `#E1DAC0` | 首屏与多数区块；穿插纯白区块做节奏 |
| 主文字 ink | `#181D27` / `#252B37` | 近黑带蓝 |
| **品牌强调紫** | `#644FB8`（rgb 100,79,184） | 强调词、连接线缆、图标点缀 |
| 道具深绿 | 收纳盒等插画里的墨绿 | 暖底上的冷色平衡 |
| 主按钮 | 近黑 `#131007` + 白字，**圆角 8px 矩形** | ⚠️ 注意是**圆角矩形不是胶囊**（matrix 是 999px 胶囊） |
| 次按钮 | 白/暖米底 + 深字 | "Join our Discord" 等 |
| 暗色 footer | 近黑墨绿 | 见下文 |

整体是**暖中性底 + 单一紫色强调**，比 matrix 多了温度和记忆点。

## 四、核心 UI 元素

1. **Hero**：左文右图布局。左：系统无衬线大标题（含紫色斜体强调词）+ 副文 + 两枚按钮（近黑实心 + 白描边）+ 一行小字信任点（"5 分钟接入 · 无需信用卡 · 70%+ 问题自动解决"）+ "Supported" 渠道图标。右：3D 黏土插画。
2. **3D 黏土插画**：台灯 / 写字板 / 绿色收纳盒 / 钢笔 / 渠道图标药丸 + 紫色线缆，暖光柔阴影，claymorphism 风。
3. **Trusted by 信任背书行**：底部一排合作方 logo（灰度低调）。
4. **按钮**：圆角 8px 矩形，主近黑次描边，部分带 `→` 箭头。
5. **卡片**：右侧用圆角卡展示产品界面（工单/总结/优先级标签等），配彩色状态标签。
6. **暗色 footer + 超大发光 logo 字**：见技巧。

## 五、用到的关键技巧

### 1. Scrollytelling 滚动叙事（和 matrix 同款 scroll-spy！）
- "Up and running in three steps" 区块：**左侧 sticky 目录** `01 Connect / 02 Detect / 03 Handoff`，**右侧分段内容卡**随滚动划过，**滚到哪步左侧对应项就高亮**（截图 `*-03.png` 里 `03 Handoff` 高亮）。
- 与 matrix 的实现思路一致：sticky 左栏 + `IntersectionObserver` scroll-spy。**说明这套"左目录右滚动"是这个赛道落地页的通用范式。**

### 2. 单词级斜体衬线强调
- 系统无衬线正文里，靠"一个紫色斜体衬线词"制造视觉焦点和品牌调性，成本极低、效果好。

### 3. 3D 黏土插画替代真实摄影
- matrix 用真实摄影/视频，lucius 用 3D claymorphic 插画——更可控、更暖、更"产品化"，适合解释抽象的"多渠道接入"。

### 4. 眼标 + 编号叙事
- `— How it works` / `— Case studies` 前导短横眼标 + `/01 /02 /03` 编号，强化分步骤的阅读节奏。

### 5. 暖底穿插白/暗区块控制节奏
- 暖米底为主，穿插纯白区块和一个近黑 footer，靠明度变化分段。

## 六、页面底部：超大发光衬线 logo 字（big-footer-logotype）

lucius 的底部和 matrix 很不一样：

- **暗色 footer**（近黑墨绿）+ 标准链接列（Product / Company / Legal）+ 社交图标 + 版权。
- 最底部是一个**横跨整宽的超大衬线 "Lucius" 字标**，带柔和发光（glow）。这是当下流行的 **"巨型页脚 logotype"** 手法——用品牌名做收尾的视觉锤，强化记忆。
- ⚠️ 与 matrix 不同：**lucius 底部没有视频**，是纯排版的发光大字（实现更轻）。

## 七、Lucius vs Matrix 速查（都给 replica 当参考）

| 维度 | Matrix | Lucius |
|---|---|---|
| 背景基调 | 纯白 + **真实摄影/视频** | **暖橄榄米** + **3D 黏土插画** |
| 标题字体 | 整句 **Instrument Serif 衬线** | **系统无衬线** + 单词紫色斜体衬线强调 |
| 强调色 | 几乎无（黑白灰） | **紫色 #644FB8** |
| 主按钮 | **999px 胶囊** | **8px 圆角矩形** |
| 底部收尾 | **全宽循环视频**（绿色山坡） | **超大发光衬线 logo 字** |
| 共同点 | scroll-spy 滚动叙事 · 斜体衬线强调 · 近黑主按钮 · trusted-by 行 · 克制留白 | ← 同 |

> 给 replica 的启发：两家都用「克制留白 + 斜体衬线强调 + scroll-spy 叙事」这套基底；差异在**底色与主视觉**（摄影 vs 插画）和**强调色**。replica 可在这个基底上选自己的主视觉母题与强调色（见 `../../视觉方案.md`）。
