# Matrix.build 落地页视觉拆解（参考）

> 来源：中文 https://matrix.build/zh ，英文 https://matrix.build/ ，2026-06-27 实测抓取（DOM 计算样式 + 截图）。
> 中英文**同一套设计**，仅文案语言不同（英文大标题如 *Bring your first Agent Company to life.*）。本文档同时覆盖中英两版。

## 一、整体气质

一句话：**「极简留白 + 衬线大标题 + 真实风景影像」** 的高级感路线。不是常见的科技深色霓虹，而是**亮色、克制、像一本杂志/一份精致产品手册**。

- 浅色主题（纯白底 + 近黑墨色字），靠**大字号衬线标题**和**真实自然影像**（旧金山湾、绿色山坡）撑高级感。
- 大量留白，元素稀疏，节奏靠滚动叙事推进。
- 唯一的"科技感"来自：等宽小标签、卡片化的产品截图、浮动 logo 药丸。

---

## 二、字体系统（关键，最值得抄）

| 角色 | 字体 | 说明 |
|---|---|---|
| **大标题 / Display** | **Instrument Serif**（Google 免费可商用） | 英文是优雅斜体衬线（如 *Agent* 单独 `<span>` 加 italic）；中文无字形，**回退到系统衬线**（宋体感），形成「英文斜体衬线 + 中文细衬线」的混排美感 |
| **正文 / UI** | **ABC Oracle**（付费 grotesque sans） | 16px 正文，行高 1.6；导航、按钮、卡片标题都用它 |
| **小标签 / 等宽** | **DM Mono**（Google 免费） | 9–11px，**全大写 + 字间距 0.66–1.32px**，半透明墨色 `rgba(17,20,24,.46)`；用于眼标(eyebrow)、域名、代码态标签如 `MATRIX DEPLOY` / `AIVIDEOPRO.IO` |

> 复刻要点：Instrument Serif + DM Mono 都免费；中文配一个气质接近的细衬线（**Noto Serif SC / 思源宋体 Light** 或仓耳今楷）即可还原。ABC Oracle 付费，可用 **Geist / Inter / HarmonyOS Sans** 替代。

## 四、核心 UI 元素

1. **Hero 全屏动态背景**：整个首屏是一块 `<canvas>`（1440×900），渲染**缓慢飘动的云/天空**（旧金山湾 + 金门大桥）。上面居中叠衬线大标题 + 玻璃质感胶囊输入框（"输入你的邮箱"）+ 次级胶囊按钮（Download for macOS）。
2. **"内置"logo 行**：hero 底部一排合作模型 logo（Claude Code、Codex、GLM、DeepSeek、Kimi、Qwen…），灰度低调。
3. **胶囊按钮**：圆角 999px，主按钮近黑实心，次按钮描边/玻璃态。
4. **卡片化产品截图**：右侧用圆角卡片包真实产品界面截图（AI Video Pro、Stripe 付费墙等），配 DM Mono 小标签。
5. **Bento 网格**：功能/集成区用大小不一的卡片网格（Agent Revenue / Wallet / Email / Website 等）。
6. **浮动 logo 药丸**：集成区一列黑色胶囊（GitHub、Slack、Notion、Stripe、Figma、Supabase、Gmail、Linear、Discord…）层叠浮动，背景是同一套云天 canvas。
7. **3D 旋转 logo**：footer 上方有一个 210×63 的 `<canvas>` 渲染会旋转的 MATRIX 3D 标识。

## 五、用到的关键技巧

### 1. Scrollytelling 滚动叙事（你问的"左目录右滚动"效果）
- **左侧 sticky 目录栏**（`.uclog-rail`，`position:sticky; top:92px`）固定不动。
- **右侧分段内容**（`.uclog-tl-text`）随滚动逐段划过，每段配 DM Mono 眼标 + 编号（01 / 02 / 03…）+ 一张产品卡。
- 滚到哪一段，**左侧对应标题就高亮变黑**（其余灰），这叫 **Scroll-spy / 滚动监听导航**，靠 `IntersectionObserver` 实现。
- 整组讲完后，sticky 元素随父容器边界"解除吸附"，整块一起上移——这是 sticky 的自然行为。
- 见截图 `02-scrollspy-1.png`（"部署"高亮）、`03-scrollspy-2.png`。

### 2. Canvas 动态背景（替代静态大图）
- Hero 和集成卡的"图片"其实是 `<canvas>` 实时渲染的飘云动画，比静态图更有生命力，也比视频更轻。
- ⚠️ 静态截图里 canvas 背景常画不出来（见 `06-integration-canvas.png` 集成卡背景是空白的，实际有飘云）。

### 3. 编号时间线（01/02/03）
- 左侧一条竖线 + 圆点节点 + 大号 DM Mono 序号，强化"一步步发生"的叙事感。

### 4. 真实影像锚定情绪
- 不用插画/3D 抽象，而是**真实自然摄影/视频**（海湾日光、绿色山坡）做首尾，传达"真实、可信、宁静的力量"。

## 六、⭐ 页面底部的"动态图片"= 一个循环视频

你注意到的底部那张"会动的图"，**本质是一个全宽循环 `<video>`，不是图片**：

- 资源：`https://static.matrix.build/brand.mp4`（1440×600，已下载存档为 **`07-footer-brand.mp4`**）。
- 内容：一片**绿色山坡/草地的舒缓风景**，缓慢运镜，循环播放（`loop`、`muted`、`autoplay`）。
- 作用：作为 footer 全宽背景，上面叠最终 CTA「让你的第一家 agent company 活起来。」+ 邮箱胶囊 + 页脚链接（隐私政策/服务条款）。
- 与 hero 首尾呼应：开头蓝天海湾（canvas），结尾绿色山坡（video），形成"晨景→暮景"的情绪闭环。

> 复刻成本提示：用循环静音视频做 footer 背景，记得 `playsinline muted loop autoplay` + 一张 poster 首帧兜底，移动端注意流量与自动播放策略。
