01 / 创建分身

3 分钟，造一个有你人格的分身

填好岗位、角色与人格设定，上传你的文档建起专属知识库——一个会按你的口吻、你的判断说话的数字分身，就此成形。

02 / 替你答重复问题

重复的问题，它替你接住

同事随时来问，分身基于你的知识库人格化作答，并标注出处。那些一天被打断十几次的重复问题，不必每次都惊动你。

03 / 答不出，诚实转你

拿不准，它不会乱答

检索不到或没有把握时，它会如实说明、把问题转交给你本人，而不是抛给同事一个看似可信的错误答案。你补答的每一句，都会沉淀回你自己的分身——下次它就答得出。

04 / 越用越懂你

答过的、聊过的，它都记得

聊过的偏好、答过的判断、谁问过什么，都会沉淀成多层记忆。它记得你，也记得每个提问的人——越用越准，越用越像你本人。

— 常见问题

动手前，
最想知道的几件事。

Replica 和普通问答机器人有什么不同？
普通机器人是一个全公司共享的 AI，学的是"公司知道什么"。Replica 是每位员工各自一个有人格的分身，按本人的口吻和判断回答，还会越用越懂你、可随岗位交接——是"每个人是谁、会怎么答"。
分身答不出来时会怎样？
检索不到或没把握时，分身不会编造：它坦白说不确定，把问题转交给你本人并记为「待回答」。你补答的每一句都会沉淀回你自己的分身，下次同样的问题它就答得出。
它会不会乱编、答错？
回答基于你上传的知识库做检索，并标注出处，可追溯；低置信时宁可转人工也不硬答。
员工离职了，他的分身怎么办？
一键把分身的知识与记忆整包移交给接替的同事，离职、休假、轮岗都不再断档——这正是 Replica 想解决的核心场景之一。
接入要多久？需要 IT 部署吗？
3 分钟自助创建：填好岗位与人格、上传文档即可用，无需漫长的部署周期。
数据安全吗？知识会泄露给别人吗？
分身与知识归属于组织，按组织 / 项目组 / 角色做权限隔离，只有被授权的人能问到对应分身。
