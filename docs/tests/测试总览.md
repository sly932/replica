## 统一运行说明

```bash
cd ~/projects/replica/web

# 单个工具
npx tsx --env-file=.env.local lib/agent/tools/__tests__/<工具名>.test.ts

# 例如
npx tsx --env-file=.env.local lib/agent/tools/__tests__/manageMemory.test.ts
```

要点：

- 测试连真实外部服务（Supabase / ofox），本地需走代理。测试文件顶部已用 `EnvHttpProxyAgent` 读取 shell 的 `HTTP(S)_PROXY`（本机 `http://127.0.0.1:7897`），运行前确保代理已开、环境变量已导出。
- 每个测试自建独立 test replica，结束在 `finally` 中删除（cascade 清理 articles/chunks/memories/conversations/knowledge_items），不污染正式数据。
- 每个测试以 `TEST PASSED ✅` 结尾、`process.exit(0)`；失败打印 `TEST FAILED ❌` 并 `exit(1)`。

## 环境依赖

| 依赖 | 用途 | 配置（web/.env.local） |
|------|------|------|
| Supabase | 数据持久化 + pgvector 检索 | `SUPABASE_URL` / `SUPABASE_SERVICE_KEY` |
| ofox embedding（OpenAI 兼容端点） | 文本向量化，1536 维 | `EMBEDDING_BASE_URL` / `EMBEDDING_API_KEY` / `EMBEDDING_MODEL` / `EMBEDDING_DIM` |
| ofox anthropic（chat 端点） | ingest 时 chunk 上下文化 | `CHAT_BASE_URL` / `CHAT_API_KEY` / `CHAT_MODEL` |
| 本地代理 | 访问外部服务 | shell `HTTP_PROXY` / `HTTPS_PROXY` |
